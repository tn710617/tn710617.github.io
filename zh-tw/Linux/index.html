DNS

顯示區域的當前 DNS 紀錄

此指令會以含有前 100 筆記錄的資源記錄集，輸出 JSON 回應。您可以指定下列額外參數：

  • limit：要列出的記錄集數目上限。
  • name：只列出含有這個確切網域名稱的記錄集。
  • type：只列出這個類型的記錄。如果有這類記錄，則必須同時使用 –name 參數。
