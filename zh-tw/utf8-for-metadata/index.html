MySQL 中, 什麼是 metadata?

任何形容資料庫的都稱為 metadata, 像是, column name, database name, user names, version names, SHOW 的輸出, 還有 INFORMATION_SCHEMA table 的內容

MySQL 中, metadata 以哪種 character set 儲存?
