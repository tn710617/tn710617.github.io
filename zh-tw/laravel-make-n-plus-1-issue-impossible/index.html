# 前言

有時專案很趕, API 一次要開很多支, 常常沒注意到就產生了 Lazy loading N+1 issue
本篇文章提供一個概念性方法來完全的杜絕 N+1 issue, 不過各位朋友啊, 這只是個概念, 沒有真正實作於 Production 環境, 如果朋友不怕死你可以試試, 但出事千萬別找我啊！
