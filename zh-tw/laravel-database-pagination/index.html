# Manually Creating A Paginator

Laravel 中, Paginator 與 LengthAwarePaginator 的差異是？

Paginator 不需要知道共有幾筆資料, 所以也無法直接取得最後一筆資料
LengthAwarePaginator 需要知道共有幾筆資料
