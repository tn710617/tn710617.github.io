# 前言

學習一個框架, Ray 的想法是, 在深入理解底層實作的原理之前, 應該先知道這個框架的 使用方法; 先學習怎麼使用這個前人造的輪子, 再學習怎麼樣造一個輪子。

Laravel Broadcast 中, toOthers method 的使用情境是？

Laravel Broadcast 中, toOthers method 的用途是？
